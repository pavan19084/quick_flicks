# Quick Flicks 🎬

A feature-rich TikTok clone built with Flutter and Firebase, offering a seamless short video-sharing experience with real-time social interactions and a TikTok-like UI.

## ✨ Features

- 🔐 Authentication with Email & Password
- 📤 Uploading Videos with Captions
- 🗜️ Video Compression before upload
- 🖼️ Thumbnail Generation from Videos
- 🎞️ Displaying Videos with Captions
- ❤️ Like/Unlike Posts
- 💬 Comment on Posts
- ❤️ Like Comments
- 🔎 Search for Users
- ➕ Follow/Unfollow Users
- 📊 Display Followers, Following, Likes & Posts of a User
- 📱 TikTok-Inspired UI and Video Feed

---

## 🚀 Tech Stack

- **Client**: Flutter
- **State Management**: GetX
- **Backend**: Firebase (Auth, Firestore, Storage)
- **Architecture**: MVC (Model-View-Controller)

---

## 🛠️ Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/pavan19084/quick_flicks.git
cd quick_flicks
```

### 2. Project Setup

Ensure required services like authentication, database, and storage are configured in your backend. Add necessary platform-specific configuration files if you're running on Android or iOS.

### 3. Run the App

```
flutter pub get
flutter run
```
